Generación de resúmenes de vídeo-entrevistas utilizando redes neuronales

Página web creada para el TFG "Generación de resúmenes de vídeo-entrevistas utilizando redes neuronales" del curso 2020/2021 de la Universidad Complutense de Madrid. El código y la documentación se encuentran en GitHub

Para obtener el mejor resultado:

  • Coloca el micrófono situado lo más cerca posible de los hablantes.
  • Evita el ruido de fondo y los ecos excesivos.
  • Intenta que los hablantes hablen al mismo tiempo lo menos posible.
