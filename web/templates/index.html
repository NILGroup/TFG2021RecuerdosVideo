Generación de recuerdos a partir de vídeos utilizando redes neuronales sequence to sequence

Página web creada para el TFG "Desarrollo de un sistema para la generación de recuerdos utilizando redes neuronales sequence to sequence" del curso 2021/2021 de la Universidad Complutense de Madrid. El código y la documentación se encuentran en GitHub.

Autores: Daniel Alcázar Muñoz y Francisco Javier Lozano Hernández
Director: Alberto Diaz Esteban

Para obtener el mejor resultado:

  • Coloca el micrófono situado lo más cerca posible de los hablantes.
  • Evita el ruido de fondo y los ecos excesivos.
  • Intenta que los hablantes hablen al mismo tiempo lo menos posible.
